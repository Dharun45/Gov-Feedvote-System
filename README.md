# Gov-Feedvote-System
FeedVote is an IoT-powered citizen feedback and grievance monitoring platform that modernizes public service evaluation using ESP32 smart kiosks, anonymous feedback collection, role-based administration, geo-location tracking, and analytics dashboards to help authorities monitor and improve service quality efficiently.
